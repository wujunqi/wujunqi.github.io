基本知识

  • 尺寸屏幕

    屏幕大小，指的是屏幕对角线长度，而不是屏幕宽度或高度。单位为英寸。如iphone7屏幕为4.7英寸，三星S6屏幕为5.1英寸， 1英寸 = 2.54 厘米

  • 显示/物理分辨率

    屏幕拥有像素的总数以及屏幕水平（宽）和垂直（高）方向的像素数量。单位为像素。像素/px是组成屏幕图像的最小点，一个相对大小单位，同屏幕里越小越密集越清晰。

  • 屏幕像素密度（ppi)

    ppi指每英寸所拥有的像素数（每英寸像素），像素密度越高，代表屏幕图像显示越清晰。

计算公式

  • 视网膜屏幕

    当你拿的手机距你约25-30厘米，如果手机达到300以上， 你的眼睛将无法分辨出像素点。

  • dpi

    dpi平面设计/印刷计数单位，指每英寸所能印刷出来的网点数。衡量打印机打印精度的主要参数之一，值越大，表明打印机的打印精度越高。

页面流程图绘制

页面流程图是指产品所有页面相互间流转关系， 用于了解产品页面间的逻辑关系。通过线框图绘制成的演示版的页面流程图，可更加直观的展示产品页面逻辑，可用于向领导、客户演示，讲解。

  • 产品标准色规范
  • 控件、页面元素（包括图标、按钮等）间距、尺寸、交互规范
  • 文本字号、颜色规范

标准色规范设计

产品标准色可分为主色、辅助色、点缀色

  • 主色：主视觉，重点突出的颜色，决定产品的色调
  • 辅助色：与主色搭配的颜色，使产品颜色搭配更完美协调
  • 点缀色：点亮画面，起到强调、引导浏览作用
