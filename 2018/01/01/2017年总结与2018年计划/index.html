2017年总结与2018年计划

发表于 2018-01-01 | 热度 ℃ | 本文总阅读量 次
字数统计 774 | 阅读时长 3

昨天和女朋友去了大学同学家吃了个饭，算是搭伙一起过了个元旦，聊起了很多大学和大学考研的事情，还打了一场久违的篮球，虚得很，现在写这篇总结还全身疼痛。时间过的真快呀，真的是老了，上个篮后个仰都觉得快要闪到我这老腰了，有时候在场上能碰到一些十七八岁的小伙子，真羡慕他们，可以不觉疲惫的奔跑，我十七八可不也是这样啊！今年是比较有意义的一年，告白了20年的学生生涯（我居然上了20年学），第一次走上了真正的职场。先对今年做一些总结，然后写下18年的计划。

17年总结

  • 磕磕绊绊可算拿到学位，终于告白研究生生活

  • 毕业大典骗老爸老妈家长必须来，带爸妈在北京游玩了三天

  • 4月13日正式入职，第一次使用mac，再也不想用windows了

  • 简短的毕业旅行了一次，带女朋友去了泰山，登上泰山之巅

  • 4月18年在自如上找到了房子，有个自己一个小窝

  • 搭建了自己的博客：番茄技术小栈

  • 写了32篇博客（或者叫阅读笔记） 博客归档

  • 听了50余本书 每天听本书

  • 写了一个微信小程序

  • 女朋友来了，每天要6点起床，竟也养成10点半左右睡觉，6点准时起床的习惯

  • 追了一部剧《猎场》

  • 开始关注商业和理财，第一次买了基金，到现在为止总挣了50几块钱

  • 看完了《http详解》

  • 学完了《高性能的PHP API 接口开发》课程

  • 学完了《MySQL性能管理及架构设计》课程

  • 学完了《nginx从入门到实践》课程

  • 学完了小程序开发课程（初级和高级）

  • 开始使用docker（真他妈好用）

  • 组内进行过一次技术分享（讲的很烂）

18年计划

  • 定亲

  • 听完200本书

  • 成为全栈工程师（3年目标）

  • 写一个真正属于自己的小程序：名字叫“曼巴精神”，把nba有曼巴精神的球员都汇集的一起，收集他们的视频，在小程序上都可以观看。

  • 开发自己的一个微信公共号

  • 每天6点起床

  • 继续学习理财

  • 再学习一门语言（python）

  • 接触大数据

后记

我知道我脑海里还有很多没有去除掉的垃圾，我就必须汲取更多有益的信息来使我摆脱这种蒙昧状态；每当我想起这句话，我就十分渴求自己成为一个把目光从洞穴墙壁移向洞穴外世界的明白人；每当我想起这句话，我就知道以知识求独立的旅程是那么地诱惑我，使我昼夜难寐。我希望到我离开这个世界前知晓“我所处的世界究竟如何运作？”
