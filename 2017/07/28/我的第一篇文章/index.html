我的第一篇博客

发表于 2017-07-28 | 分类于 成长 | 阅读次数 | 本文总阅读量 次

1.我为什么要搭建自己的博客？

记得那时我还在上研究生的时候，我们实验室的一个老师说想给我们实验室搭建一个网站，问实验室的人谁有兴趣，实验室的人没有人表示愿意做，只有我回答了老师，我说，我想在暑假试试看，因为当时的我真的非常迷茫，科研搞的一塌糊涂，技术也和本科没有什么区别，我不知道我要干什么，也不知道我能干什么，当时研究生的我活的是非常痛苦，以至于我非常能理解每年北邮人研究生跳楼的事件。
那真是一段不堪回首的故事，以至于今年4月毕业的时候，当我得知我交完最后一份材料就可以毕业的时候，我是以在北交三年来最快的速度从宿舍跑到了九教，因为我知道我终于逃离了，我终于可以肆无忌惮的实习了，工作了，后来的毕业大典，因为父母来，带他们去游玩，也没有参加，跟我研究生过的太痛苦有关。接受了这个任务后，我非常的兴奋，我先在知乎上找到了搭建网站要学的知识，然后夜以继日的看书和写代码，经常是写到也里两三点（我室友对此很无语）。等我把前端、后端已经搭建网站的知识都学了一遍后，我开始实现一些小功能，看一些实战类的书，那段时间是我最快乐的时间，真的是最快乐的两个月，也是看的书最多的两个月。基础知识学习的差不多了，我发现我还是不能做出一个很好的网站。后来，我去实习了，我首先找了一家非常小的创业公司，在里面的好处是，什么都可以自己尝试，而且不用上坐班，可以在学校提交代码，并且一天大概能挣到200块钱，当时的老板是我老乡，比较器重我，培训为期2周，然后才开始正式写代码，给我们发工资，老板在我一个礼拜之后就给我安排了项目。在这个创业公司，我可以接触到网站从开始数据库设计到后台开发，甚至和客户打交道我也参与了，那段时间，我经常和我们老板去中国人民大学和那些老师接触，问他们的需求，只是那时候，因为自己主要负责后端，前端的知识没有好好学，当时真的是一个特别好的机会，因为我们的老板是一个全栈性的选手，现在真后悔没有好好跟他学前端。两个月时间，我做了两个项目，我尝试性的投递了新浪微博的简历，后来我去面试了，过了三轮面试。现在我也留在了微博，我们老大对我很好，真的特别感谢带我的第一个老板，我的所有起点都是从哪里开始的。因为来上研究生之前我真的是特别的渺茫，什么知识都懂一点，但都不精，不知道自己能干什么，通过这个小公司，我重要知道，我能靠写代码挣到钱。毕业前我尝试自己搭建一个网站来记录自己。后面因为各种原因耽搁了，前几天，这种想法又愈发的强烈。两天前，我开始行动了，终于在今天搭建完成了。搭建这个博客主要是想记录一下自己的成长和知识积累。之前看过一张学习的金字塔。

所以写技术博客还是很重要的

2.域名为什么要用：yanqieto.top

取这个域名，也和我上研究生有关，因为实在是太煎熬了，研二上学期我开始接触心理学和时间管理类的知识，对我影响最大的就是那本番茄工作法图解，我至今在用的工作法（但写论文就别用了，什么工作法都没有用），它让我真正的静下心来学习，番茄钟一响，就投入到专心的25分钟。我看完的和砖头一样大的技术书都是借助它完成的。所以我对番茄有特殊的好感（本身自己也爱吃番茄），所以就叫它了，fanqieto.top，这个域名的谐音是：番茄土豆，但精神层面上也有的：它是我番茄工作法使用的软件，fanqietotop寓意：

番茄创造巅峰

好傻的寓意

3.其他

这几天一直在搭建这个博客，真实太折腾了，熬了两个晚上的夜。所幸终于搭建好了。买了5年的域名，以后会记录自己关于人生成长方面的事情，更多的是一些写技术博文。
